# Image Classification on CIFAR- 100 dataset

### CIFAR-100 dataset: https://www.cs.toronto.edu/~kriz/cifar.html
CIFAR-100 dataset has 100 classes containing 600 images each, 500 training and 100 testing.
The 100 classes are also grouped into 20 super classes.
I trained some basic neural networks on both the super categories and the finer categories.
